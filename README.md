amwares-manage
=============
start
-------

* `install mongodb`
* `create a connection name 'report'`
* `start this project`
```
$ git clone git@github.com:timmyLan/amwares-manage.git
```
```
$ cd amwares-manage && npm install
```
```
$ npm start
```
* `create a tab http://127.0.0.1:3000 on your Browser`
